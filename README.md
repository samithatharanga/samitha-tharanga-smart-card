# 🚀 Samitha Tharanga Smart Card

Official Digital Smart Card and Portfolio of **Samitha Tharanga Wijesinghe**. 
Entrepreneur specializing in Graphic Design, UI/UX, Video Editing, and AI.

🔗 **[View Live Demo Here](https://samithatharanga.github.io/samitha-tharanga-smart-card/)**

## ✨ Features
* 📱 **Mobile-First Glassmorphism UI:** Premium, modern design using Tailwind CSS.
* 💾 **Save Contact (vCard):** One-click download to save contact details directly to the phone.
* 🔗 **Web Share API:** Native sharing capabilities on mobile devices.
* 🔍 **SEO Optimized:** Built-in JSON-LD structured data for Google Rich Results.
* ⚡ **Single File Template:** Extremely easy to host and modify (Zero setup required).

## 🛠️ How to Use (Create Your Own)
This project is 100% open-source! You can create your own premium digital business card in 2 minutes:

1. **Fork** or download this repository.
2. Open the `index.html` file.
3. Scroll to the bottom and edit the `cardData` object with your personal details, links, and profile image.
4. Host it for free on GitHub Pages!
